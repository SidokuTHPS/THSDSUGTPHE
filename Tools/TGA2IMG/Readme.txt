+++ Instructions
-----------------------------
Simply execute TGA2IMG_UG2.exe
and select the file you want
to convert. An IMG file will then
be output with the almost the
same name.

Note: The output is not the
same name to prevent acidently
overwriting original files!

Command Prompt USAGE:
TGA2IMG_UG2 [input.img] [game] [compress]
