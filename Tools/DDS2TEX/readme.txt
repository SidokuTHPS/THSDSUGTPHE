TonyHawks DDS to TEX converter
Version: 0.1
Website: http://jmn-web.tk
THPS Alias: Defeat0r

<- description
This tool is to convert the
directX .DDS format into the
THPS4/THUG/THUG2 .TEX format.

-----------------------------
+++ Instructions
-----------------------------
Simply execute DDS2TEX_TH.exe
and select the file you want
to convert. Then select the .TEX
file you wish to replace. A .DDS
file will then be output in the
current directory with a simular
filename.

Note: You only need to select a
file to replace for reading its
checksums, NO file is actually
replaced!

Note: You CAN convert multiple
.DDS files into a single .TEX by
opening a selection of .DSS files
with TEX2DDS_TH.exe.

Command Prompt USAGE:
DDS2TEX_TH [input.dds]
