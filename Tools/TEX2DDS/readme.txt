TonyHawks TEX to DDS converter
Version: 0.1
Website: http://jmn-web.tk
THPS Alias: Defeat0r

<- description
This tool is to convert the
THPS4/THUG/THUG2 .TEX format
into the directX DDS format.

-----------------------------
+++ Instructions
-----------------------------
Simply execute TEX2DDS_TH.exe
and select the file you want
to convert. A .DDS file will
then be output in the current
directory with a simular filename.

Note: I highly recommend ATI's
"TheCompressionator" software
for processing .DDS files. You
can download this FREE from
"http://www.ati.com/developer/compressonator.html".

Command Prompt USAGE:
TEX2DDS_TH [input.tex]
